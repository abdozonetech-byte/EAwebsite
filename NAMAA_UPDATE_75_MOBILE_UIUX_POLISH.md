# Namaa AI Update 75 — Mobile UI/UX Polish

## Goal
Make the full Namaa flow feel stable and professional on phones without changing the desktop design or backend logic.

## Improved
- Mobile off-canvas agents sidebar with backdrop blur.
- Safer `100dvh` viewport handling for modern mobile browsers.
- Better safe-area spacing for iOS/Android browser bars.
- Compact topbar, chat bubbles, composer and buttons on small screens.
- Better mobile layout for Namaa Design workspace.
- Better mobile browser preview for Namaa Website.
- Better mobile strategy boards and Namaa branding visibility.
- Better email popup bottom-sheet behavior on phones.
- Reduced-motion support.

## Not changed
- Gemini routing.
- Agent flow order: Talk → Design → Website → Strategy.
- CRM/Sheet final update remains disabled for now.
- No API keys added to the frontend.
