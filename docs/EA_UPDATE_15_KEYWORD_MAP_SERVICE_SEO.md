# EA Update 15 — Keyword Map + Service Page SEO Optimization

Date: 2026-06-21

## Goal

Implement the next SEO step after the technical SEO foundation: assign one clear search intent and one main keyword cluster to every priority page.

This follows Google Search Central principles from the SEO Starter Guide:

- make the site easier for search engines to understand;
- organize related pages clearly;
- use descriptive URLs and breadcrumbs;
- write unique and concise titles/descriptions;
- connect related pages with links;
- create useful, people-first content.
