# Namaa AI Update 34 — Project Factory UX Map

This update keeps Update 33 as the stable intelligence checkpoint and adds the first UX map for the next Namaa Project Factory direction.

- Two clear entry modes: **Free Talk** and **Build My Project**.
- Sidebar Agent Hub language for Namaa Talk, Namaa Images, and NamaaDev.
- Factory route map: Talk → Strategy → Design → Web.
- Hero mode cards explaining the difference between free conversation and guided project creation.
- Initial agent map cards for Strategy Agent, Design Agent, and Web Agent.
- New flow actions for Free Talk and Build Project.
- Health endpoint update: `34-project-factory-ux-map`.

## Not changed

- Gemini Talk / Images / Dev endpoints stay untouched.
- Existing PDF, mockup, and dev generation logic remains compatible.
