Upgrade 11 — Big Mobile Responsive Polish

Goal:
Create a cleaner, more organized mobile experience across the Elboubakry website without changing the desktop UI or global brand identity.

What changed:
- Added assets/css/elboubakry-upgrade-11-mobile-responsive.css.
- Linked the new mobile CSS layer to all HTML pages.
- Improved mobile page rhythm, section spacing, container padding, typography scale, and tap targets.
- Reorganized homepage mobile flow line-by-line: hero, growth system, funnel, systems carousel, FAQ, footer.
- Converted the complex orbit growth system into a clear vertical system-card layout on mobile only.
- Improved hero mobile readability: name, role, focus line, CTA buttons, social icons, and portrait placement.
- Improved portfolio carousel on mobile: one clean swipe card per view, snap scrolling, duplicate hidden slides removed from mobile view.
- Improved FAQ mobile organization: cleaner accordion spacing and full-width CTA.
- Improved Insights listing mobile layout: hero, keywords, dashboard, and article cards stack cleanly.
- Improved article pages mobile reading: tighter hero, readable content width, stacked CTA buttons, one-column cards and sidebar.
- Improved diagnostic page mobile layout: stacked actions, readable cards, comfortable spacing.
- Improved footer mobile stacking and trust badges.

Important restrictions respected:
- No desktop redesign.
- No change to brand colors, identity, analytics, SEO, security, or lead form logic.
- No fake content added.
- No section removed.
