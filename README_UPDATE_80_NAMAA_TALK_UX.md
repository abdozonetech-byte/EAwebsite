# Update 80 — Namaa Talk UX + Language + Personality Fix

## What changed

- Reduced Namaa Talk questions: no more long questionnaire.
- Project mode now asks only essential info first:
  1. Project name
  2. Short project description / offer
  3. City or market
- Namaa confirms understanding before opening Namaa Design.
- Greeting now shows two clear choices:
  - Free Talk
  - Build Project
- Improved language behavior:
  - Arabic script input → Moroccan Darija in Arabic script
  - Darija Latin input → Namaa can ask if user prefers Arabic-script Darija or Darija Latin
  - No unnecessary language switching
- Namaa Talk now avoids raw Gemini style and keeps a short, friendly Moroccan business tone.
- Frontend message bubbles now render bold choices and line breaks cleanly.

## Flow remains

Namaa Talk → Namaa Design → Namaa Website → Namaa Strategy

CRM / Sheet / Pack Link remains parked for the final update.
