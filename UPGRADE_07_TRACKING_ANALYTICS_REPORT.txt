How to activate GA4
1. Open assets/js/analytics.js.
2. Replace:
   GA4_MEASUREMENT_ID: 'G-XXXXXXXXXX'
   with your real GA4 ID, for example:
   GA4_MEASUREMENT_ID: 'G-ABC123DEF4'
3. Keep COOKIE_STORAGE false for a privacy-first cookieless setup, or set true only if you add a proper cookie/privacy notice.
4. Deploy and check GA4 Realtime.
